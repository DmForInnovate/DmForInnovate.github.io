Hexo搭建Blog

安装node.js，Hexo，Git

  • 安装node.js
  • 安装Hexo：或在node.js下使用命令npm install hexo-cli -g安装
  • 安装Git：Git使用教程

初始化Hexo

  • 新建一个文件夹Blog（名字随你取）这是博客目录
  • 打开node.js黑窗窗，cd 进入Blog目录
  • 初始化：命令hexo init
  • 执行 命令 npm install
    • 初始化后在Blog目录下会生成以下文件
  • 执行命令hexo s -g
    • 如图
  • 在浏览器地址栏输入http://localhost:4040/
    • 你将看到
    • 太丑，显然连微操都打不到 next–

下载主题

  • next主题github地址：here
  • 第一次搭建推荐使用 next主题
  • 其他主题下载：here
  • 下载后解压到 Blog>theme目录下

主题配置

  • Next主题
    • this mytheme
    • next主题详解 上面写的很详细

部署到github

  • 创建github账号
  • 新建一个仓库 命名：用户名.github.io
  • 打开Blog下的 config.yml文件
    • 配置deploy：字段
    • deploy:
      type: git
      repo: git@github.com:DmForInnovate/DmForInnovate.github.io.git
      这里的repo名要设为自己创建GitHub时的用户名，我的是DmForInnovate
  • 下载自动发布工具
    • 在Blog目录 git bash
    • 命令：npm install hexo-deployer-git --save
  • 输入命令：hexo clean && hexo g && hexo d
  • 在浏览器地址栏输入 用户名.github.io
    这里的用户名就是你的GitHub用户名
